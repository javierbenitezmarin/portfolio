# Communication

## Approach

1. **Read the audience.** An executive, a PM, and an engineer need the same truth in
   different words. Match the vocabulary to the profile and drop jargon that doesn't serve
   the listener.
2. **Lead with the point.** State the decision or the ask first, then the reasoning.
   Respect people's time.
3. **Be honest, especially about bad news.** Surface risks, blockers, and uncertainty
   early. No surprises.
4. **Close the loop.** Confirm understanding, write it down, and follow up.

## Principles

- Honesty over comfort. A hard truth delivered kindly beats a comfortable omission.
- Clarity is a feature. If the listener didn't get it, the message failed — not the listener.
- Tailor, don't dumb down. Adjust the framing, keep the substance.
